# Parking Space Availability 🚗🅿️

This project predicts and displays available parking spaces using data analysis and machine learning. It helps drivers quickly find parking spots, reduces traffic congestion, and improves parking management in smart cities.

## Features
- Predict parking space availability
- User-friendly interface
- Data visualization of parking occupancy
- Helps reduce traffic and save time

## Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn
- Matplotlib / Seaborn

## How to Run
1. Clone the repository
2. Install required libraries
3. Run the Streamlit app
